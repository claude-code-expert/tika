# agents/ - 에이전트 프롬프트 디렉토리

특정 작업 유형에 대해 Claude Code가 참조하는 에이전트 프롬프트를 관리하는 디렉토리이다. 에이전트는 반복적이고 정형화된 구현 작업을 일관된 패턴으로 수행하도록 가이드한다.

## 하위 문서

### todo-crud.md
티켓(TODO) CRUD 기능 구현 에이전트. 새로운 CRUD 기능 추가 시 따라야 하는 구현 레이어 순서(타입 → 스키마 → 검증 → 쿼리 → API → 훅 → UI → 테스트), position 계산 규칙, 상태 전환 비즈니스 로직을 정의한다.

## 새 에이전트 추가 방법

1. 이 디렉토리에 `에이전트이름.md` 파일을 생성한다.
2. 에이전트의 목적, 구현 순서, 규칙, 비즈니스 로직 등을 마크다운으로 기술한다.
3. Claude Code에게 해당 에이전트를 참조하여 작업하도록 요청한다.
