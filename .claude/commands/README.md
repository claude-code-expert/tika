# commands/ - 슬래시 명령어 디렉토리

Claude Code에서 슬래시 명령어(`/`)로 실행할 수 있는 명령어 정의 파일을 관리하는 디렉토리이다.

## 사용 방법

Claude Code 대화창에서 `/project:명령어이름` 형태로 실행한다.

```
> /project:deploy     # Vercel 배포 실행
> /project:migrate    # DB 마이그레이션 실행
```

## 하위 문서

### deploy.md
Vercel 배포 명령어. 배포 전 체크리스트(빌드, 린트, 테스트 확인)와 배포 실행 단계, 배포 후 확인사항을 정의한다.

### migrate.md
Drizzle ORM 마이그레이션 명령어. 스키마 수정 → 마이그레이션 생성 → 적용의 워크플로우와 주의사항을 정의한다.

## 새 명령어 추가 방법

1. 이 디렉토리에 `명령어이름.md` 파일을 생성한다.
2. 마크다운 형식으로 명령어의 단계와 실행할 명령을 기술한다.
3. Claude Code에서 `/project:명령어이름`으로 실행할 수 있다.
