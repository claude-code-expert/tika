# Tika — Workspace 티어 요구사항

> **버전**: Workspace (v0.2.x)
> **레포지토리**: tika (현재 레포)
> **작성일**: 2026-03-11
> **범위**: Personal 전체 + 팀 워크스페이스 협업 기능
> **제외 항목**: 스프린트, 이메일 초대, MCP, 리치 텍스트 에디터, 온프레미스

---

## 1. 티어 개요

| 항목 | 내용 |
|------|------|
| 대상 | 개인 사용자 + 소규모 팀 |
| 팀 워크스페이스 | 최대 **3개** |
| 워크스페이스당 티켓 | 최대 **1,000개** |
| 인증 | Google OAuth (NextAuth.js v5) |
| 초대 방식 | 링크 초대 (7일 유효) / 참여 신청 |
